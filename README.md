# Twitter Junkie

X (旧Twitter) のブラウジング体験をより快適にするための、非公式Chrome拡張機能です。
メディアの閲覧やタイムラインの消化に特化した、便利なスクロール機能と画面最大化機能を提供します。

## 主な機能

1. **自動スクロール＆直感的な速度調整**
   - パネルのボタン、またはマウスとキーボードのショートカットから自動スクロールを開始できます。
   - **直感的なホイール操作モード**: 「右クリック＋マウスホイール」だけで、スクロールの方向と速度をシームレスにコントロールできます。
   - 独自のキーバインド設定にも対応しています（Spaceキーで停止など）。

2. **動画の最大化（シアターモード）**
   - タイムライン上の動画に「最大化ボタン」を追加し、ワンクリックで画面いっぱいに拡大表示します。
   - 動画の枠外（すりガラス部分）をクリックするか、Escキーを押すことでいつでも元に戻れます。
   - お好みに合わせて、最大化ボタンをドラッグ＆ドロップで好きな位置に移動させることも可能です。

3. **邪魔にならないUI設計**
   - X公式の画像・動画の全画面表示モードに入った際は、自動スクロール用のUIパネルが自動的に隠れます。
   - パネル自体を「常に非表示」にする設定も用意されています（ショートカットキーのみで操作したい方向け）。

4. **日英バイリンガル対応**
   - オプション画面から、表示言語を「日本語」と「English」で切り替えることができます。

## 導入方法（インストール）

この拡張機能は Chrome ウェブストアには公開されていません。以下の手順でローカル環境から直接インストールしてご使用ください。

1. このリポジトリのファイル一式をダウンロード（ZIPでダウンロードして解凍、または `git clone`）します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して拡張機能の管理画面を開きます。
3. 画面右上にある **「デベロッパー モード」** のスイッチをオンにします。
4. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. ダウンロード（解凍）した `Twitter junkie` のフォルダを選択します。
6. Chromeに拡張機能が追加され、X (twitter.com / x.com) を開くと機能が有効になります。

## 設定の変更

Chromeのツールバーにある拡張機能アイコン（パズルピースのマーク）から「Twitter Junkie」を選択し、**「設定 (Options)」** を開くことで、ショートカットキーや機能のオン/オフを自由にカスタマイズできます。

## デスクトップ用アプリ・スマホでの利用について

現状、PC版ブラウザ (Google Chrome 等の Chromium 系ブラウザ) のみをサポートしています。

## 免責事項

本拡張機能は非公式ツールです。X (旧Twitter) の仕様変更により、一部機能が突然動作しなくなる可能性があります。あらかじめご了承ください。
